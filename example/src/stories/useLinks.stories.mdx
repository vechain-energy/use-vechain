# useLinks


<Meta title="Hooks/useLinks" />

Generate a link to the explorer to allow users to check on transaction details themself. The link is for the current network configuration.  
Default links can be overwritten by defining them in the `options` of the `VeChainProvider`:

The option values are:

```json
{
  "linkTransaction": "https://explore-testnet.vechain.org/transactions/{txId}",
  "linkAccount": "https://explore-testnet.vechain.org/accounts/{address}",
  "linkBlock": "https://explore-testnet.vechain.org/blocks/{blockId}"
}
```

Function | Example | Note
:------- | :------ | :--------
`getTransactionLink` | `getTransactionLink(txId)` | returns a link to the network explorer for a transaction id
`getAccountLink` | `getAccountLink(address)` | returns a link to the network explorer for an account
`getBlockLink` | `getBlockLink(blockId)` | returns a link to the network explorer for a block id

export const GetTransactionLink = () => {
  const { getTransactionLink } = useLinks()
  return <a href={getTransactionLink('0x0026890dfbaf80ba08bc4f5d3e8f3a0a81b4f5a17398cc11fe304118d6634a43')} target='_blank' rel='noreferrer'>{getTransactionLink('0x0026890dfbaf80ba08bc4f5d3e8f3a0a81b4f5a17398cc11fe304118d6634a43')}</a>
}
